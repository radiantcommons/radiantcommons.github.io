the mission

Radiant Commons is a non-stock, non-profit corporation enabling financial privacy everywhere, by extending the Penumbra ecosystem.

Radiant focuses on four main activities:

  • We coordinate with external partners for middleware and ecosystem tooling (like block explorers and API interfaces) and advise the Penumbra ecosystem on grants.
  • We contribute upstream to Penumbra and Cosmos through an in-house tech team.
  • We help users govern the protocol by building, managing, and researching open source coordination mechanisms, while actively moderating and participating in Penumbra governance.
  • Finally, we incubate teams building privacy-first applications atop Penumbra, with the goal of spinning them out into independent entities.
